# PhasOverlay – Ghost Data

This repo holds the live ghost roster for **PhasOverlay**.

`ghosts.json` is fetched by the app every time the Evidence window opens. Edit it here
and the change reaches everyone within a few minutes — no app update or reinstall needed.

**To edit:** open `ghosts.json`, click the pencil (✏️) icon, make your change, and
commit. Keep the JSON valid — if it's broken, the app safely ignores it and keeps the
last good copy.
